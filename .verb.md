## Usage

```js
var Emitter = require('{%= name %}');
```


## API

### Emitter(obj)


As a mixin:

```js
var Emitter = require('emitter');
var user = { name: 'tobi' };
Emitter(user);

user.emit('im a user');
```

As a prototype mixin:

```js
var Emitter = require('emitter');
Emitter(User.prototype);
```
